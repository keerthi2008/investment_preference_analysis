# Investment Preferences Analysis in Excel

## Objective
Analyze participants' investment preferences and understand how investments are distributed across different avenues using Excel.

## Tools Used
Microsoft Excel

## Analysis Performed
- Used Pivot Tables to summarize investment choices.
- Analyzed investment avenues such as Mutual Funds, Equity Market, Government Bonds, and Fixed Deposits.
- Identified reasons for investment including Better Returns, Capital Appreciation, High Interest Rates, and Risk-Free options.
- Created charts to visualize the distribution of investment preferences and reasons for investment.

## Outcome
The analysis provides insights into participants' preferred investment options and the factors influencing their investment decisions.
